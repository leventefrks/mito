Sziasztok!

A feladat megoldását a dist mappában találjátok minify-olva a HTML-t és a hozzá tartozó CSS-t is.
Amennyiben a forrást megtekintenétek a src mappában találjátok.

ITCSS-t és BEM-et használtam SCSS pre-processorral.
GULP task runner segített a HTML,SASS minify-olásában, illetve transformálásában.

Az alábbi Gulp plugin-eket használtam fel a feladat megoldása során:
- gulp-minifycss
- gulp-concat
- gulp-prefix
- gulp-sass
- gulp-htmlmin
- gulp-browser-sync

Az oldal fő struktúráját CSS GRID-el készítettem el, az elemeket flexbox-al helyeztem el.

A böngésző, amit használtam: 
Google Chrome v.66.0.3359.117
